# 项目规则

## 概述
- 现在需要开发一个企业级管理系统的脚手架模板，脚手架模板的主要功能包括用户管理、角色管理、菜单管理、操作日志管理、参数配置管理等功能。

## 编写语言
- 项目使用 UI 前端框架为 Art Design Pro 在编码前需要查看相关的参考文档。
- 项目使用 Spring Boot 编写后端代码, 后端代码需要符合 RESTful 风格。
- 与账号相关的所有功能需要使用 sa-token 框架进行权限验证
- 项目使用 MySQL 数据库, 数据库表需要符合命名规范。
- 需要前后端分离。

## 数据库连接
- 项目使用 MySQL 数据库, 数据库连接需要在 application.properties 中配置。
- 数据库连接配置需要包含数据库 URL、用户名、密码等信息。

```
用户名：root
密码：151505
数据库 URL：jdbc:mysql://localhost:3306/acc
```
## 规范与原则
- 使用插件加快速开发的。
- 编码遵循阿里云编码规范, 包括命名规范、代码格式、注释规范等。
- 项目使用 Maven 进行依赖管理, 所有依赖需要在 pom.xml 中配置。
